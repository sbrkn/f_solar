# Commit Message Standards

## General Guidelines
1. Use the imperative mood in the subject line.
2. Limit the subject line to 50 characters.
3. Separate the subject from the body with a blank line.

## Structure
- **Subject**: A brief summary of the changes.
- **Body**: Detailed explanation of the change, justification, and any references.

## Example
```
Add new feature to improve user experience

This feature enhances user experience by... 
```
